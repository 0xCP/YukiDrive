# YukiDrive

一个 Onedrive &amp; SharePoint 文件浏览程序，支持国际版和世纪互联版。

后端采用 .net core 3.1，前端使用 Vue ，前后端分离，无刷新加载。

无需搭建运行环境，下载并配置完成后直接运行。
